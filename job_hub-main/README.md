# JobHub - Professional Placement Portal

## About

JobHub is a comprehensive campus placement platform connecting students, universities, and companies. It features smart job matching, automatic domain-based verification, scoped analytics, and role-based dashboards.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd jobhub

# Install dependencies
npm install

# Start the development server
npm run dev
```
